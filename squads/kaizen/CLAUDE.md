# Kaizen Squad — Continuous Improvement Engine

## Single Entry Point

```
@kaizen-chief *analyze
```

O squad que vigia e melhora todos os outros. Auditoria em 6 dimensões, análise semanal automatizada, máx. 5 recomendações priorizadas por relatório.

---

## Análise em 6 Dimensões

| Dimensão | Agente | O que detecta |
|----------|--------|---------------|
| Estrutura (Topology) | `@topology-analyst` | Redundâncias, gaps de cobertura |
| Performance | `@performance-tracker` | Lead time, rework rate, throughput |
| Gargalos | `@bottleneck-hunter` | Constraints via TOC de Goldratt |
| Competências | `@capability-mapper` | Gaps de skill via Wardley Maps |
| Ferramentas | `@tech-radar` | Radar ADOPT/TRIAL/ASSESS/HOLD |
| Custos | `@cost-analyst` | ROI, custo por squad, custo por entrega |

## Comandos

| Comando | O que faz |
|---------|-----------|
| `*analyze` | Análise completa — 6 dimensões, todos os agentes |
| `*gaps` | Detecta competências e ferramentas faltantes |
| `*performance` | Dashboard de métricas (DORA, OKR) |
| `*radar` | Atualiza tech radar (Adopt/Trial/Assess/Hold) |
| `*cost` | Análise de custo por squad, ROI |
| `*report` | Relatório semanal — máx. 5 recomendações |
| `*self-improve` | O squad analisa e melhora a si mesmo |

## Agentes

| Agente | Mente Clonada | Papel |
|--------|---------------|-------|
| `@kaizen-chief` | — | Orquestrador semanal |
| `@topology-analyst` | Skelton & Pais | Topologia e redundâncias |
| `@performance-tracker` | Forsgren + Croll | Métricas DORA/OKR |
| `@bottleneck-hunter` | Goldratt | Theory of Constraints |
| `@capability-mapper` | Wardley + Bersin | Wardley Maps de competências |
| `@tech-radar` | Fowler + Ford | Radar de tecnologias |
| `@cost-analyst` | Kaplan/Norton + Doerr | ROI e OKRs de custo |

## Cadência

**Execução:** Semanal (domingo 20:00 BRT) via `*analyze`
**Auto-melhoria:** Mensal via `*self-improve`
**Recomendações:** Máx. 5 por relatório, priorizadas por Evidence/Impact/Action/Cost/ROI
