# Curator Squad — Video Content Curation

## Single Entry Point

```
@curator-chief {descrição ou transcrito}
```

Curator recebe a missão e roteia automaticamente pelo pipeline de curadoria.

**Princípio fundamental: Zero Invention — o squad NUNCA inventa texto. Apenas monta conteúdo existente.**

---

## Pipeline Principal

```
Phase 1: MINE         Phase 2: CATALOG       Phase 3: NARRATIVE
ATHENA 3-pass    →    Timestamp catalog  →   Arc building
MQR scoring           moments.md             miner → writer

Phase 4: FORMAT       Phase 5: DELIVER
Cut scripts      →    Editor guide
YAML + MD             FFMPEG cuts
```

## Comandos

| Comando | O que faz |
|---------|-----------|
| `*mine` | Mineração ATHENA 3-pass + scoring MQR |
| `*catalog` | Catálogo de timestamps com momentos scored |
| `*narrative` | Construção de arco narrativo (Ken Burns, McKee) |
| `*format` | Formatar em long, shorts, ou multi-format |
| `*cut-script` | Gerar cut script editor-ready (YAML + guia) |
| `*validate` | Rodar 5 quality gates (blocking) |

## Quality Gates (Todos Bloqueantes)

- QG-001: Qualidade de transcrição (mín. 80% precisão)
- QG-002: Qualidade de mineração (momentos.md + anotações)
- QG-003: Qualidade narrativa (arco completo, sem lacunas)
- QG-004: Qualidade de output (timestamps exatos, guia completo)
- QG-005: Curadoria de dados (dados reais, zero invenção)
