# Checklist de Completude: Relatório Estratégico (UC-AEPC-001, 003, 004, 005)

Use este checklist para validar o Relatório Estratégico Completo antes de entregar ao cliente.

## Pipeline Completo Executado

- [ ] UC classificado pelo `chefe-estrategico` (QG-AEPC-001)
- [ ] Tipo de ação e fase identificados pelo `classificador-civel` (QG-AEPC-002)
- [ ] 6 eixos CPC auditados pelo `auditor-processual` (QG-AEPC-003)
- [ ] Peças extraídas (8 categorias) pelo `leitor-pecas-civel`
- [ ] Pesquisa CPC/STJ (5 dimensões) pelo `pesquisador-cpc`
- [ ] Análise estratégica (5 partes + 3 cenários) pelo `estrategista-civel`
- [ ] Análise recursal pelo `analista-recursos` (UC-AEPC-001 e 003)
- [ ] Plano de ação (4 seções) pelo `orientador-civel`
- [ ] Relatório salvo via Write pelo `redator-estrategico` (QG-AEPC-004)

## Conteúdo do Relatório

- [ ] Seção 1: Identificação do Processo
- [ ] Seção 2: Resumo Executivo (3-5 pontos-chave)
- [ ] Seção 3: Diagnóstico Processual com tabela de riscos e artigos CPC
- [ ] Seção 4: Fundamentos Jurídicos (CPC + STJ + Súmulas)
- [ ] Seção 5: 3 cenários com probabilidades (soma = 100%)
- [ ] Seção 5: Viabilidade de acordo com justificativa
- [ ] Seção 6: Análise recursal com checklist de admissibilidade (se aplicável)
- [ ] Seção 7: Ações urgentes (até 7 dias) e cronograma
- [ ] Seção 8: Conclusões e Próximos Passos

## Qualidade Técnica

- [ ] Todos os artigos CPC corretos e numerados
- [ ] Teses repetitivas STJ identificadas quando aplicável
- [ ] Cenários com percentuais somando 100%
- [ ] Prazos CPC com artigos específicos
- [ ] Bloco `citacoes` com pelo menos 3 entradas
- [ ] Nome do arquivo seguindo padrão: `relatorio-estrategico-civel-[slug]-[AAAA-MM-DD].md`
