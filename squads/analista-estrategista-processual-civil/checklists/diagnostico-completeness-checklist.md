# Checklist de Completude: Diagnóstico Processual (UC-AEPC-002)

Use este checklist para validar o Relatório de Diagnóstico antes de entregar ao cliente.

## Classificação (chefe-estrategico + classificador-civel)

- [ ] UC-AEPC-002 classificado pelo `chefe-estrategico`
- [ ] Tipo de ação identificado (conhecimento/execução/cautelar)
- [ ] Fase processual atual mapeada
- [ ] Polo ativo e passivo identificados (ou marcados [NÃO IDENTIFICADO])
- [ ] Competência (estadual/federal, vara) indicada

## Auditoria CPC (auditor-processual)

- [ ] 6 eixos auditados: pressupostos processuais, condições da ação, prescrição/decadência, preclusão, nulidades, prazos imediatos
- [ ] Cada risco acompanhado do artigo CPC/CC específico
- [ ] Riscos CRÍTICOS destacados
- [ ] Prazos fatais próximos sinalizados com ALERTA

## Relatório (redator-estrategico)

- [ ] Relatório em MODO_DIAGNOSTICO (estrutura simplificada)
- [ ] Seção de riscos imediatos com artigos CPC
- [ ] Prazos urgentes listados
- [ ] Recomendação rápida (1-3 ações prioritárias) presente
- [ ] Relatório salvo via Write (confirmar caminho do arquivo)
- [ ] Bloco `citacoes` presente

## Qualidade Geral

- [ ] Sem opinião estratégica após receber diagnóstico (escopo restrito ao UC-AEPC-002)
- [ ] Linguagem acessível para o cliente
- [ ] Artigos CPC corretos e relevantes
