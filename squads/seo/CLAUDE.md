# SEO Expert Squad — Post-Design SEO Optimization

## Single Entry Point

```
@seo-chief *seo-evaluate
```

Otimização SEO pós-design para sites construídos com AIOS. Avalia (0-100), otimiza, e reporta — como Yoast/RankMath mas como agentes inteligentes.

---

## Pipeline de 3 Fases

```
Phase 1: AUDIT          Phase 2: OPTIMIZE       Phase 3: REPORT
Score 0-100        →    Apply all fixes    →    Before/After
(all 8 agents)          (agents fix)             comparison
```

## Comandos

| Comando | O que faz |
|---------|-----------|
| `*seo-evaluate` | Auditoria completa (score 0-100, 7 categorias) |
| `*seo-optimize` | Aplicar todos os fixes identificados na auditoria |
| `*seo-report` | Relatório comparativo antes/depois com delta |
| `*seo-full` | Pipeline completo: avaliar → otimizar → reportar |
