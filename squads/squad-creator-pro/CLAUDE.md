# Squad Creator Pro — Upgrade Pack

## O que é

O Squad Creator Pro é um **upgrade pack** que transforma o Squad Creator base em uma fábrica de squads de elite. Não substitui o base — adiciona 3 especialistas que o `@squad-chief` delega.

**Requer:** `squads/squad-creator/` instalado

```
@squad-chief *create-squad {domínio}
```

Com o Pro instalado, o squad-chief detecta automaticamente e delega para os especialistas Pro.

---

## O que o Pro adiciona

| Funcionalidade | Base | Pro |
|----------------|------|-----|
| Criação de squad | Template-driven | Mind cloning + DNA extraction |
| Fidelidade | 60-75% | 85-95% |
| Agentes especializados | 0 extra | 3 (oalanicolas, pedro-valerio, thiago_finch) |
| Axioma assessment | ✗ | ✓ (score ponderado por dimensão) |
| Model routing | ✗ | ✓ (Opus/Sonnet/Haiku por tipo de task) |

## Agentes Pro

| Agente | Especialidade |
|--------|---------------|
| `@oalanicolas` | Mind architecture — extrai Voice DNA + Thinking DNA |
| `@pedro-valerio` | Process design — SOPs e frameworks operacionais |
| `@thiago_finch` | Business strategy — framing e posicionamento |

## Fluxo com Pro

```
@squad-chief recebe: *create-squad {domínio}

Com Pro instalado:
1. @thiago_finch — framing de negócio e posicionamento
2. @oalanicolas — pesquisa elite minds, extrai DNA (Voice + Thinking)
3. @pedro-valerio — design de processos e SOPs
4. @squad-chief — monta squad com fidelidade 85-95%
5. Axioma Assessment — score por dimensão, PASS/FAIL
```
