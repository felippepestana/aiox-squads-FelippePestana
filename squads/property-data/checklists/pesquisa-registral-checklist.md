# Checklist: Pesquisa Registral Completa

**Aplicável a:** UC-PD-001, UC-PD-ALL
**Agentes validadores:** pesquisador-registral, property-data-chief

## Dados da Matrícula

- [ ] Número da matrícula identificado
- [ ] Cartório de registro identificado
- [ ] Cadeia dominial mapeada (histórico de proprietários)
- [ ] Área do terreno registrada
- [ ] Área construída registrada (se houver)
- [ ] Confrontações descritas
- [ ] Ônus reais verificados (hipoteca, alienação fiduciária, penhora)
- [ ] Averbações listadas

## Certidões

- [ ] Certidão de habite-se localizada (número, data, órgão)
- [ ] Certidão negativa de ônus reais consultada
- [ ] IPTU consultado (inscrição, valor venal, situação)

## Dados Cadastrais

- [ ] Inscrição municipal identificada
- [ ] Área conforme cadastro municipal
- [ ] Uso registrado no cadastro municipal

## Situação Fiscal

- [ ] IPTU em dia ou débitos identificados
- [ ] Dívida ativa verificada
- [ ] Débitos municipais consultados

## Validação de Qualidade

- [ ] Todas as fontes citadas com URL/referência e data
- [ ] Dados cruzados com documentos do leitor-documental (se aplicável)
- [ ] Divergências sinalizadas explicitamente
- [ ] Campos não encontrados marcados como [NÃO ENCONTRADO]
