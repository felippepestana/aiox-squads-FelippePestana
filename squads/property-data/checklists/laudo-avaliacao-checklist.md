# Checklist: Laudo de Avaliação Imobiliária (ABNT NBR 14653)

**Aplicável a:** UC-PD-006, UC-PD-ALL
**Agentes validadores:** avaliador-imovel, relator-imobiliario, property-data-chief

## Identificação

- [ ] Solicitante identificado
- [ ] Finalidade da avaliação declarada
- [ ] Data-base da avaliação explícita
- [ ] Imóvel completamente identificado (endereço, matrícula, inscrição)

## Caracterização do Imóvel

- [ ] Área do terreno informada (com fonte)
- [ ] Área construída informada (com fonte)
- [ ] Tipo e padrão construtivo classificados
- [ ] Estado de conservação classificado (Ross-Heidecke a-h)
- [ ] Idade aparente estimada
- [ ] Materiais predominantes descritos

## Caracterização da Região

- [ ] Uso predominante do entorno descrito
- [ ] Infraestrutura urbana avaliada
- [ ] Tendência de mercado indicada
- [ ] Zoneamento identificado

## Análise Visual

- [ ] Fotos analisadas (quando fornecidas)
- [ ] Geolocalização verificada
- [ ] Grau de confiança atribuído às observações
- [ ] Cruzamento visual vs matrícula realizado

## Método de Avaliação

- [ ] Método declarado com justificativa
- [ ] Método adequado ao tipo de imóvel e dados disponíveis

## Pesquisa de Mercado (Método Comparativo)

- [ ] Mínimo 3 dados de mercado comparáveis
- [ ] Cada dado com: endereço, tipo, área, valor, fonte, data
- [ ] Dados de ofertas com fator oferta aplicado
- [ ] Dados recentes (preferencialmente últimos 12 meses)

## Homogeneização

- [ ] Fatores de homogeneização declarados e justificados
- [ ] Fator oferta aplicado (quando dados de anúncio)
- [ ] Fator localização justificado
- [ ] Fator área aplicado
- [ ] Fator padrão aplicado
- [ ] Fator idade/conservação aplicado

## Depreciação (Método Evolutivo)

- [ ] CUB utilizado com mês/estado correto
- [ ] Vida útil estimada compatível com tipo de construção
- [ ] Coeficiente Ross calculado
- [ ] Coeficiente Heidecke atribuído conforme estado de conservação
- [ ] Depreciação total calculada

## Especificação ABNT

- [ ] Grau de fundamentação declarado (I, II ou III)
- [ ] Grau de precisão declarado (I, II ou III)
- [ ] Intervalo de confiança calculado (quando Grau II ou III)
- [ ] Requisitos mínimos do grau atendidos

## Resultado

- [ ] Valor unitário (R$/m²) apresentado
- [ ] Valor total do imóvel apresentado
- [ ] Limites inferior e superior informados
- [ ] Ressalvas e limitações documentadas

## Qualidade do Laudo

- [ ] Todas as fontes rastreadas no bloco de citações
- [ ] Referências normativas listadas (NBR 14653-1, 14653-2, 12721)
- [ ] Nenhum valor emitido sem método e fundamentação
- [ ] Arquivo salvo via Write
