## Estrutura do Loop (90 segundos)

```
[VÍDEO 15s]  →  [SLIDE 1 — Gancho/Problema: 15s]
             →  [SLIDE 2 — O que é: 15s]
             →  [SLIDE 3 — Como funciona: 15s]
             →  [SLIDE 4 — Resultados: 15s]
             →  [SLIDE 5 — CTA: 15s]
             →  [VÍDEO 15s]  →  LOOP ∞
```

**Transições:** cross-dissolve 1s entre slides | fade para branco 1s vídeo → slides
