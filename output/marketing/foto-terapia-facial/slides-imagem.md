# Sequência de Slides — Foto Terapia Facial
**Formato:** 16:9 | **Total:** 5 slides × ~15s = 75 segundos | **Uso:** Loop recepção (TV)

---

## DIRETRIZES GERAIS DOS SLIDES

| Parâmetro             | Valor recomendado                                              |
|-----------------------|----------------------------------------------------------------|
| Tempo por slide       | 12–15 segundos (com 1s de transição cross-dissolve)           |
| Máximo de palavras    | 12–15 palavras por slide                                       |
| Tamanho de fonte      | Título ≥ 60pt · Subtítulo ≥ 44pt (legível a 3m)              |
| Fundo                 | Foto clínica premium com overlay escuro 40% ou fundo branco   |
| Paleta                | Branco, azul-luz (#4FC3F7), dourado suave (#D4AF37)           |
| Transição             | Cross-dissolve 1s — sem transições bruscas                    |

---

## SLIDE 1 — GANCHO / PROBLEMA
**[ 15s – 30s do loop total ]**

**Fotografia sugerida:**
- Rosto de mulher com manchas, textura irregular ou pele sem viço
- Close nos olhos com linhas finas ao redor
- OU: paciente olhando para o espelho com expressão pensativa

**Texto na tela:**
```
┌─────────────────────────────────┐
│                                 │
│   Manchas, pele apagada         │
│   ou rugas finas te incomodam?  │
│                                 │
└─────────────────────────────────┘
```
*Título: bold, branco sobre fundo escuro ou faixa colorida*

**Animação de entrada:** Fade-in do texto (0,5s delay após foto aparecer)

---

## SLIDE 2 — O QUE É
**[ 30s – 45s do loop total ]**

**Fotografia sugerida:**
- Painel LED sendo aproximado do rosto (vista lateral elegante)
- OU: imagem técnica do equipamento, clean, sobre fundo branco
- OU: luz colorida do LED refletindo sobre a pele

**Texto na tela:**
```
┌─────────────────────────────────┐
│                                 │
│   Foto Terapia Facial           │
│   Luz LED que regenera          │
│   sua pele de dentro para fora  │
│                                 │
└─────────────────────────────────┘
```
*Ícone pequeno de onda de luz ao lado do título é opcional*

---

## SLIDE 3 — COMO FUNCIONA
**[ 45s – 60s do loop total ]**

**Fotografia sugerida:**
- Close da pele sob o LED (luz visível, pele com brilho)
- OU: imagem conceitual de luz penetrando camadas da pele
- OU: paciente em repouso durante o procedimento (expressão serena)

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   A luz estimula colágeno,           │
│   reduz manchas e revitaliza a pele  │
│                                      │
│   ✓ Indolor   ✓ Sem agulhas          │
│   ✓ Sem downtime                     │
│                                      │
└──────────────────────────────────────┘
```

---

## SLIDE 4 — RESULTADO / BENEFÍCIOS
**[ 60s – 75s do loop total ]**

**Fotografia sugerida:**
- Pele radiante, uniforme, iluminada (close de alta qualidade)
- OU: comparativo discreto antes/depois (com autorização)
- OU: paciente sorrindo com pele visivelmente mais luminosa

**Texto na tela:**
```
┌─────────────────────────────────┐
│                                 │
│   Pele mais firme,              │
│   uniforme e luminosa           │
│                                 │
│   Resultado visível             │
│   desde a 1ª sessão             │
│                                 │
└─────────────────────────────────┘
```

---

## SLIDE 5 — CTA + GANCHO
**[ 75s – 90s do loop total ]**

**Fotografia sugerida:**
- Ambiente da clínica: recepção aconchegante, sala de tratamento ou equipe
- OU: paciente saindo satisfeita da consulta (expressão de leveza)
- Tons quentes, luz suave — transmite acolhimento

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   Transforme sua pele ainda hoje     │
│                                      │
│   Agende sua avaliação gratuita      │
│   📲 [CONTATO]                       │
│                                      │
│   [NOME DA CLÍNICA]                  │
└──────────────────────────────────────┘
```
*Logotipo da clínica no canto inferior direito em todos os slides*

---

## FLUXO COMPLETO DO LOOP

```
[VÍDEO 15s] ──fade──► [SLIDE 1: 15s] ──dissolve──► [SLIDE 2: 15s]
    ──dissolve──► [SLIDE 3: 15s] ──dissolve──► [SLIDE 4: 15s]
    ──dissolve──► [SLIDE 5 CTA: 15s] ──fade──► [VÍDEO 15s] → LOOP
```
**Duração total do loop: ~90 segundos**
