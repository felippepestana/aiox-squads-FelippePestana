# Sequência de Slides — Aplicação de Botox
**Formato:** 16:9 | **Total:** 5 slides × ~15s = 75 segundos | **Uso:** Loop recepção (TV)

---

## DIRETRIZES GERAIS DOS SLIDES

| Parâmetro             | Valor recomendado                                              |
|-----------------------|----------------------------------------------------------------|
| Tempo por slide       | 12–15 segundos (com 1s de transição cross-dissolve)           |
| Máximo de palavras    | 12–15 palavras por slide                                       |
| Tamanho de fonte      | Título ≥ 60pt · Subtítulo ≥ 44pt (legível a 3m)              |
| Fundo                 | Foto clínica premium com overlay escuro 40% ou fundo branco   |
| Paleta                | Branco, bege-nude (#F5E6D3), dourado suave (#D4AF37)          |
| Transição             | Cross-dissolve 1s — sem transições bruscas                    |

---

## SLIDE 1 — GANCHO / PROBLEMA
**[ 15s – 30s do loop total ]**

**Fotografia sugerida:**
- Close no rosto de mulher com rugas de expressão na testa ou ao redor dos olhos
- Expressão natural — não exagerada
- OU: espelho refletindo expressão pensativa com foco nas linhas de expressão

**Texto na tela:**
```
┌─────────────────────────────────┐
│                                 │
│   As rugas de expressão         │
│   te fazem parecer mais velho   │
│   do que você se sente?         │
│                                 │
└─────────────────────────────────┘
```
*Título: bold, branco sobre fundo com overlay escuro*

---

## SLIDE 2 — O QUE É
**[ 30s – 45s do loop total ]**

**Fotografia sugerida:**
- Ampola de toxina botulínica sobre superfície médica limpa (elegante, não assustadora)
- OU: profissional segurando o produto com luvas, ambiente clínico premium
- OU: close nas mãos habilidosas do profissional preparando o procedimento

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   Botox relaxa os músculos           │
│   que formam rugas de expressão      │
│                                      │
│   Resultado: aparência natural       │
│   e descansada                       │
└──────────────────────────────────────┘
```

## SLIDE 3 — COMO FUNCIONA
**[ 45s – 60s do loop total ]**

**Fotografia sugerida:**
- Profissional aplicando com delicadeza (foco na técnica, não na agulha)
- Paciente com expressão tranquila durante a aplicação
- OU: ilustração elegante dos pontos de aplicação no rosto (estilo diagrama médico clean)

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   Procedimento rápido                │
│   20 a 30 minutos                    │
│                                      │
│   ✓ Mínimo desconforto               │
│   ✓ Sem downtime                     │
│   ✓ Retorno imediato às atividades   │
│                                      │
└──────────────────────────────────────┘
```

---

## SLIDE 4 — RESULTADO / BENEFÍCIOS
**[ 60s – 75s do loop total ]**

**Fotografia sugerida:**
- Rosto de paciente com aparência descansada e natural pós-procedimento
- Expressão genuína — sorriso suave, confiante
- OU: antes/depois discreto e elegante (com autorização)
- Iluminação: natural, suave, sem filtros exagerados

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   Aparência mais jovem               │
│   e descansada — com naturalidade    │
│                                      │
│   Resultado em 7 a 14 dias           │
│   Duração: 4 a 6 meses              │
│                                      │
└──────────────────────────────────────┘
```

## SLIDE 5 — CTA + GANCHO
**[ 75s – 90s do loop total ]**

**Fotografia sugerida:**
- Paciente satisfeita saindo da clínica ou em consulta de avaliação
- OU: profissional sorrindo com paciente — transmite relação de confiança
- Ambiente clean, cores neutras, luz natural

**Texto na tela:**
```
┌──────────────────────────────────────┐
│                                      │
│   Você já está no lugar certo        │
│                                      │
│   Pergunte sobre Botox               │
│   na sua consulta hoje               │
│                                      │
│   Ou agende: 📲 [CONTATO]           │
│                                      │
│   [NOME DA CLÍNICA]                  │
└──────────────────────────────────────┘
```
*Nota: o gancho "você já está no lugar certo" aproveita o contexto da sala de espera*

---

## FLUXO COMPLETO DO LOOP

```
[VÍDEO 15s] ──fade──► [SLIDE 1: 15s] ──dissolve──► [SLIDE 2: 15s]
    ──dissolve──► [SLIDE 3: 15s] ──dissolve──► [SLIDE 4: 15s]
    ──dissolve──► [SLIDE 5 CTA: 15s] ──fade──► [VÍDEO 15s] → LOOP
```
**Duração total do loop: ~90 segundos**
